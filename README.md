# Codewars Challenges

Daily coding challenges to sharpen problem-solving skills and deepen understanding of core programming concepts.

<h2>How It's Made:</h2>  
<p>Tech used: JavaScript</p>

<p>Each challenge is approached by first breaking the problem into smaller, manageable steps, then implementing a solution using modern JavaScript features, recursion, and ES6+ syntax.</p>

<p>I focus on writing clean, readable code first, then iterating to improve efficiency and elegance.</p>

<h2>Optimizations</h2>

<p>After getting a working solution, I revisit problems to:</p>

<ul>
<li>Reduce time and space complexity</li>
<li>Refactor imperative code into more declarative patterns</li>
<li>Explore alternative approaches (e.g., using reduce instead of loops)</li>
<li>Compare my solutions with top Codewars submissions to learn new techniques</li>
</ul>

<h2>Lessons Learned:</h2>

<ul>
<li>There are often multiple valid solutions—readability and efficiency both matter</li>
<li>Strong fundamentals (loops, conditionals, array/object manipulation) are everything</li>
<li>Small syntax improvements (like destructuring or chaining) can significantly clean up code</li>
<li>Thinking in patterns makes future problems easier to solve</li>
<li>Debugging is a skill just as important as writing the initial solution</li>
</ul>
